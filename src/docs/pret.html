OpenGNTG for starters

Canvas

Change color

To change the color of your canvas, select the canvas tool. In the property bar, change the color to your needs.

Clear canvas

Clearing the canvas will delete all the shapes. To do that, select File > New File.

Basic Shapes

Line

To create a line, select the line tool. Draw the line in the canvas by clicking and dragging.

Square

To create a square, select the square tool. Draw the square in the canvas by clicking and dragging.

Rectangle

To create a rectangle, select the rectangle tool. Draw the rectangle in the canvas by clicking and dragging.

Polygon

To create a polygon, select the polygon tool. Click on the canvas to start drawing the polygon. Click on the canvas again until it forms the desired shape. Click on the first vertex to close the polygon and finish drawing.

Color

You can set the color of your shape by changing the color in the property bar before drawing.

Alternatively, you can change the color of an already existing shape by selecting it in the object bar and change the color inside the property bar.
